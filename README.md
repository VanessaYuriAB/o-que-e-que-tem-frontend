# 🌱🥣 O que é que tem? Na sopa, creme ou patê | Foodtech sustentável (conceitual)

Aplicação web full stack que simula um modelo de cozinha sustentável (delivery & drive-thru) baseado
no aproveitamento de alimentos próximos ao vencimento provenientes de mercados parceiros.

A plataforma propõe um sistema de assinatura flexível, no qual os usuários podem escolher os pratos
a cada consumo, conforme a disponibilidade de um cardápio dinâmico, reduzindo o desperdício
alimentar e incentivando o consumo consciente.
